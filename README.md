# 德州扑克管理 - 网页版

这是一个德州扑克游戏的管理应用，从微信小程序版本转换为纯 HTML/CSS/JavaScript 网页版本。

## 功能特性

- **游戏大厅**: 显示所有活跃游戏，支持创建新游戏
- **游戏管理**: 创建游戏、添加玩家、记录赏金事件
- **游戏结算**: 自动计算积分排名、奖励分配、费用分摊

## 核心玩法

1. **创建游戏**: 设置游戏参数（玩家人数、入场费、赏金规则等）
2. **选择玩家**: 从预设玩家列表中选择，或添加自定义玩家
3. **游戏进行**: 记录玩家的最终积分和赏金事件
4. **结算计算**: 自动计算每个玩家的收益和支出

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage (数据存储)

## 使用方法

1. **启动应用**: 直接在浏览器中打开 `index.html`
2. **创建游戏**:
   - 点击"创建新游戏"
   - 填写游戏名称和参数
   - 选择参与玩家
3. **管理游戏**:
   - 添加玩家
   - 记录赏金事件（谁猎杀谁）
4. **结算游戏**:
   - 确认最终积分
   - 设置吃饭费用
   - 查看结算结果

## 数据存储

所有数据存储在浏览器的 LocalStorage 中，包括：

- 游戏信息
- 玩家数据
- 赏金记录
- 结算结果

## 部署说明

这是一个纯静态网页应用，可以部署到任何支持静态文件的服务上：

- GitHub Pages
- Netlify
- Vercel
- 普通 Web 服务器

## 主要差异（与微信小程序版本）

- 使用 LocalStorage 替代 wx.setStorageSync
- 使用 sessionStorage 传递页面间数据
- 使用原生 DOM 操作替代微信小程序框架
- 使用浏览器 alert/confirm 替代微信小程序模态框
- 使用标准 HTML 表单控件

## 浏览器兼容性

支持所有现代浏览器：

- Chrome 60+
- Firefox 60+
- Safari 12+
- Edge 79+

## 开发说明

如需修改或扩展功能：

- `storage.js`: 数据存储逻辑
- `styles.css`: UI 样式
- 各页面 JS 文件: 业务逻辑

数据会持久化保存在浏览器中，如需重置可以清除 LocalStorage。
